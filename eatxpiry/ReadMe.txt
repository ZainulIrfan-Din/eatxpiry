Pre-Run requirements:
1) Download XAMPP
2) Move eatxpire_PHP folder into htdocs folder in XAMPP folder.
3) Run xampp-control.exe
4) Run Apache and MySQL
5) Create a database in localhost with these details:
	a) Database name: loginregister
	b) Table name: users
	c) 5 attributes: id, fullname, username, password, email
